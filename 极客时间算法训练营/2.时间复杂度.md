# 2. 时间复杂度

## 2.1 常见的时间复杂度
* O(1): 常数复杂度
* O(lgn): 对数复杂度
* O(n): 线性时间复杂度
* O(nlgn): ngln
* O(n^2): 平方
* O(2^n): 指数
* O(n!): 阶乘


**O(1) < O(n) < O(nlgn) < O(n^2) < O(2^n) < O(n!)** 


## 2.2 主定理

| 算法 | 公式 | 运行时间 |
|----|----|----|
| 二分查找 | T(n) = T(n/2) + O(1) | O(lgn) |
| 二叉树遍历 | T(n) = 2T(n/2) + O(1) | O(n) |
| 最优矩阵搜索 | T(n) = 2T(n/2) + O(lgn) | O(n) |
| 归并排序 | T(n) = 2T(n/2) + O(n) | O(nlgn) |
